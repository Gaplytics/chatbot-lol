This is a test environment for the Gaply AI Chatbot Widget.

Integration Instructions

The floating widget should appear in the bottom right corner of this page automatically.

It fetches its token from the local Docker stack running on port 8080.

Test Checklist:

  • Click the widget to open the chat panel.
  • Verify you hear the "Welcome" audio (if configured in agent.py).
  • Send a text message and wait for an answer.
  • Click the Mic icon to speak to the agent.
