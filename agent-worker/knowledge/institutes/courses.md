# Gaplytiq Institute Courses

We offer domain-specific courses for different educational backgrounds.

## Tech Track
Eligible streams: **B.Tech, M.Tech, BCA, MCA**

Tech track students get access to:
- Tech specific learning courses
- Coding Tests (Python, DSA, SQL)
- Tech Interview Preparation
- HR Interview Preparation

## Non-Tech Track
Eligible streams: **BBA, MBA**

Non-Tech track students get access to:
- Non-Tech specific learning courses
- Domain/Functional MCQs
- Domain Interview Preparation (Functional + Case Study)
- HR Interview Preparation
