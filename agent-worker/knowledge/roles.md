# User Roles and Permissions

Gaplytiq Institute has specific user roles that dictate dashboard access and test capabilities.

## Admin
- Has full platform control.
- Manages users.
- Configures tests (toughness, timing, questions).
- Handles approval workflows.

## TPO (Training & Placement Officer)
- Enters batch details: Year, Stream, Strength.
- Configures tests on the basis of stream (Tech vs Non-Tech).

## Students (Tech Stream: B.Tech, M.Tech, BCA, MCA)
- Dashboard access.
- Access to Tech Courses.
- Access to Tech Test Modules (Coding, AI/ML RAG).
- Tech Interview and HR Interview preparation.

## Students (Non-Tech Stream: BBA, MBA)
- Dashboard access.
- Access to Non-Tech Courses.
- Access to Non-Tech Test Modules (Functional Test).
- Domain Interview and HR Interview preparation.
